## 主要发现

### 1. 理论检验：优序融资理论得到强支持

**核心证据：**
- **基准模型（M1）**：NPR系数 = -0.6095***（p<0.001），符号为负
- **交互固定效应（M1'）**：NPR系数 = -1.1774***，效应更强
- **分组回归（M2）**：国企系数-0.7184***，民企系数-0.5143***，均为负
- **交互项模型（M3）**：国企边际效应-0.7716，民企-0.5785，均为负

**结论**：所有模型中NPR系数均显著为负，一致支持优序融资理论——盈利能力越强的公司，杠杆率越低。该结论在不同产权性质企业中均成立。

### 2. 产权性质的重要调节作用

**关键发现：**
- 国有企业平均杠杆率（0.485）显著高于民营企业（0.363），差异达0.121***
- 国企NPR系数绝对值（-0.7184）显著大于民企（-0.5143），Z检验p=0.0002
- 交互项显著（NPR×SOE = -0.1931**），证实产权性质调节效应

**制度解释**：
- 国企软预算约束：虽融资能力强，但受政策驱动去杠杆影响更明显
- 民企融资摩擦：信息不对称严重，股权融资更倾向用于经营扩张而非偿债

### 3. NPR-Lev关系的时序结构性变化

**转折点识别：**
- 2012-2014年：系数约-0.15至-0.27（负效应较弱）
- **2015年**：系数骤降至-0.48（关键转折点）
- 2016-2024年：持续下降至-0.92（负效应增强3倍）

**政策背景**：2015年股灾及后续去杠杆政策强化，企业更倾向用股权融资替代债务，放大了NPR对杠杆率的负向影响。

### 4. 企业规模的非线性调节效应

**函数系数模型（M5）发现：**
- **拐点**：Size ≈ 22.5（对应总资产约50亿元）
- **小规模企业**（Size < 22.5）：负效应随规模增大而减弱（-1.47 → -0.50），支持信息不对称假设
- **大规模企业**（Size > 22.5）：负效应随规模增大而增强（-0.50 → -3.99），反映政策压力和规模效应

**门槛模型（M6）**：最优门槛值γ = -0.10（去均值后），与M5拐点区域基本吻合，支持非线性调节效应。

### 5. 宏观货币政策的显著影响

**M2增长率的负向效应**：
- M1'模型中M2_growth系数 = -0.0059***（p<0.001）
- 与传统预期相反，可能反映：
  - 样本期内M2高增长常伴随信贷窗口指导等紧缩政策
  - 货币宽松期企业更倾向股权融资（如定向增发）而非加杠杆
  - 去杠杆政策抵消了货币扩张的杠杆效应

## 研究贡献与政策启示

### 理论贡献
1. **优序融资理论的强证据**：使用2011-2025年中国A股大样本数据，通过多种模型设定（TWFE、IFE、分组回归、交互项）一致验证了优序融资理论
2. **产权制度的调节作用**：首次系统揭示了国企与民企在资本结构决策上的系统性差异，丰富了转型经济体资本结构理论
3. **时序结构性变化**：识别出2015年为关键转折点，为理解制度变革对企业融资行为的影响提供了新证据

### 政策启示
1. **差异化监管**：国企与民企的融资行为差异显著，政策制定应考虑产权性质的异质性响应
2. **去杠杆政策的优化**：2015年后NPR负效应持续增强，表明股权融资在降杠杆中发挥重要作用，应继续完善股权融资市场
3. **中小企业融资支持**：小规模企业（Size < 22.5）信息不对称严重，需加强信息披露和融资担保机制建设
