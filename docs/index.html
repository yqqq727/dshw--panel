上市公司资本结构影响因素分析

基于2011-2025年中国A股上市公司面板数据的实证研究

作者：曾悦晴 CHAN YUE QING

日期：2026年4月24日

📖 研究概述

本研究基于 2011-2025 年中国 A 股上市公司面板数据，系统检验了资本结构的影响因素，主要回答以下问题：

  • 🎯 理论检验：A 股上市公司资本结构更符合权衡理论还是优序融资理论？
  • 🏢 产权性质：国有企业和民营企业的资本结构决策是否存在显著差异？
  • 📈 时序演变：NPR-Lev 关系是否随时间发生结构性变化？
  • 🏭 企业规模：企业规模如何调节 NPR 对杠杆率的影响？
  • 💰 宏观政策：货币供应量增长如何影响企业资本结构？

🎯 主要发现

1. 优序融资理论得到强支持

核心证据：

  • M1: NPR 系数 = -0.6095***（p < 0.001）
  • M1': NPR 系数 = -1.1774***
  • M2a (国企): -0.7184***
  • M2b (民企): -0.5412***

结论：所有模型中 NPR 系数均显著为负，一致支持优序融资理论。

2. 产权性质的显著调节作用

关键数据：

  • 国企平均杠杆率：0.4850
  • 民企平均杠杆率：0.3636
  • 差异：0.1214***

制度解释：国企软预算约束 + 政策驱动去杠杆 vs 民企融资摩擦

3. 2015 年为关键转折点

时序变化：

  • 2012-2014 年：系数约 -0.23
  • 2015 年：系数骤降至 -0.48
  • 2016-2024 年：持续下降至 -0.92

政策背景：2015 年股灾及去杠杆政策强化

4. 企业规模的非线性调节

函数系数模型（M5）：

  • 拐点：Size ≈ 22.5（总资产约 50 亿元）
  • 小规模企业：负效应随规模增大而减弱 ✅
  • 大规模企业：负效应随规模增大而增强

5. 宏观货币政策的显著影响

M1' 模型：M2_growth 系数 = -0.0059***

与传统预期相反，可能反映去杠杆政策抵消了货币扩张效应。

📊 数据说明

数据来源

  • 数据库：CSMAR
  • 样本期间：2011-2025 年
  • 最终样本：144,960 个观测值，4,433 家公司

变量定义

变量 定义 计算方法
Lev 资产负债率 总负债/总资产
NPR 净利润率 净利润/总资产
Size 企业规模 ln(总资产)
SOE 国有企业标识 股权性质判断

🎓 研究结论与展望

理论贡献

  1. 优序融资理论的强证据：使用中国 A 股大样本数据，通过多种模型设定一致验证
  2. 产权制度的调节作用：揭示国企与民企融资决策的系统性差异
  3. 时序结构性变化：识别 2015 年为关键转折点

政策启示

  1. 差异化监管：国企与民企融资行为差异显著，政策应考虑异质性
  2. 去杠杆政策的优化：股权融资在降杠杆中发挥重要作用
  3. 中小企业融资支持：小规模企业信息不对称严重，需加强支持
